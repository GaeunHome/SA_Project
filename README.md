## 功能

### 階級
- 達到一定碳權點數可升級，並顯示階級圖片於首頁。
- 部分商品需達到對應等級才可兌換。

### 碳權存摺
- 使用者可通過此頁面查看碳權使用紀錄。

### 傳值
- AI 智慧洗或其他系統可透過 Rest API Framework 傳值進入本系統，並且顯示在碳權存摺。
